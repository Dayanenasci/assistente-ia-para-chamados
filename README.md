# Assistente de IA para Abertura de Chamados

Projeto conceitual que simula a utilização de um assistente de IA para apoio na abertura e organização de chamados técnicos, utilizando linguagem natural, automação e governança.

## Objetivo

Facilitar a comunicação entre usuários e o time de Engenharia, garantindo solicitações mais claras, estruturadas e rastreáveis.

## Tecnologias Envolvidas

- SharePoint
- Power Automate
- IA generativa (estilo Copilot)
- Governança de processos# assistente-ia-para-chamados
